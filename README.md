# Lovable Production Site

This is the production deployment repository for the Lovable website, automatically deployed to GitHub Pages.

## Live Site

Once GitHub Pages is enabled, the site will be available at:
**https://esaino.github.io/lovable-site/**

## Deployment

This repository uses GitHub Actions to automatically build and deploy to GitHub Pages on every push to the main branch.

## Technology Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
